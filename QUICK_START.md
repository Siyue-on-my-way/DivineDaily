# Divine Daily 快速参考

## 🚀 快速启动

```bash
cd /mnt/DivineDaily
./restart.sh
```

## 📋 常用命令

| 操作 | 命令 | 说明 |
|------|------|------|
| 🔄 重启服务 | `./restart.sh` | 停止并重新启动所有服务 |
| 🛑 停止服务 | `./stop.sh` 或 `./stop.sh -a` | 停止所有服务 |
| 📝 查看日志 | `./logs.sh` 或 `./logs.sh backend` | 查看服务日志 |
| ✅ 检查配置 | `./check.sh` | 检查环境和配置 |
| 🚀 快速启动 | `./start.sh` | 简单启动（旧版） |
